# Volatile

Volatile is a modern implementation of the classic **8-puzzle sliding game**, built using **React Native and Expo**. The objective is to arrange the numbered tiles in order by sliding them into the empty space. While the rules are simple, solving the puzzle efficiently requires logic, planning, and strategy.

The app focuses on a clean interface, smooth animations, and responsive gameplay to provide a simple yet engaging puzzle experience.

---

## Features

- Classic **3×3 sliding puzzle gameplay**
- Multiple **difficulty levels** (Easy, Medium, Hard)
- **Move counter** to track performance
- **Timer** to measure completion speed
- Smooth **tile animations**
- Minimal and distraction-free UI
- Supports **Android and Web**

---

## Tech Stack

- **React Native**
- **Expo**
- **Expo Router**
- **TypeScript**
- **React Native Animated API**
